# Frontend — POC

React + TypeScript + Vite frontend covering the 5 user stories. Every interactive element has a `data-testid` attribute consumed by the Selenium tests generated by the pipeline.

## Structure

```
src/
  types.ts              # Shared TypeScript types and Portuguese label maps
  api.ts                # Typed fetch wrappers for every backend endpoint
  auth.tsx              # AuthProvider + useAuth hook (localStorage-backed)
  pages/
    Login.tsx           # US-01: login form with lockout feedback (423 + Retry-After)
    Register.tsx        # US-02: registration form with redirect on success
    RequestsList.tsx    # US-04: list + filters; US-05: cancel confirmation dialog
    CreateRequest.tsx   # US-03: new request form with client-side validation
  App.tsx               # BrowserRouter + route guards (AuthGuard, GuestGuard)
  main.tsx              # React entry point
  index.css             # Global styles
```

## Running

Via Docker:

```bash
docker compose up -d frontend
```

Locally (requires pnpm):

```bash
pnpm install
pnpm dev
```

Open `http://localhost:5173`. The backend must be running on `http://localhost:8001`.

## Routes

| Route | Page | Guard |
|---|---|---|
| `/login` | Login | guest (redirects authenticated users → /requests) |
| `/register` | Register | guest |
| `/requests` | RequestsList | auth (redirects unauthenticated → /login) |
| `/requests/new` | CreateRequest | auth |
| `*` | — | redirects → /requests |
